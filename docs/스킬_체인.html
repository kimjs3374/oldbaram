oldbaram 스킬 발동 체인 (v6-edge)

게이머 관점 설명 · 조건 충족 순간 → 스킬 큐 등록 → 스케줄러가 1회 시전 → 상태 유지 중엔 반복 안 함

전체 흐름

① 내 HP/MP OCR 읽음 (0.5초마다) → ② 값이 조건 만족하는 순간 감지 → ③ 스케줄러에 스킬 이름 등록 → ④ 순서대로 1회 시전 → ⑤ 조건 풀렸다가 다시 만족하면 또 발동

스킬 9종 발동 체인

💀 자가부활 NumPad 6
1
언제 발동?
내가 죽었을 때 (HP = 0)
2
어떻게 감지?
HP OCR 결과가 0 으로 딱 바뀌는 순간
3
시전 동작
TAB → HOME → TAB (나한테 타겟)
→ 메인힐 토글 OFF
→ 부활 버튼 0.5초 연타
→ 자힐 1초 연타
→ ESC → TAB → TAB (격수 복귀)
→ 메인힐 토글 ON
재발동: HP 가 0 초과로 복구된 뒤 또 0 되면 다시 발동
❤️ 자힐 NumPad 1 (메인힐 재사용)
1
언제 발동?
내 HP % 가 설정 임계치 미만일 때 (예: 50% 아래)
2
어떻게 감지?
HP OCR 결과가 임계치 위 → 아래로 딱 내려가는 순간
3
시전 동작
자가부활과 동일한 블록 A+B 시퀀스
(부활 연타 + 자힐 연타 한 세트)
재발동: HP 가 임계치 이상으로 회복된 뒤 또 밑으로 내려가면 다시 발동
💙 공력증강 NumPad 3
1

MP OCR 결과가 임계치 위 → 아래로 딱 내려가는 순간

재발동: MP 가 임계치 이상으로 회복된 뒤 또 밑으로 내려가면 다시 발동
🆘 격수부활 NumPad 6
1

격수가 죽었고 (HP=0) + 내가 살아있을 때
2
어떻게 감지?
격수 PC 가 UDP 로 보내주는 HP 가 0 으로 딱 바뀌는 순간

NumPad6 을 1.5초 동안 0.15초 간격으로 연타

🔮 파혼술 NumPad 7
1

격수 PC 의 버프 OCR → UDP 로 전송 →

3

NumPad7 을 0.5초 동안 0.1초 간격으로 연타

⏱️ 파력무참 NumPad 8
1
언제 발동?
게임 내장 쿨타임 180초마다 (무조건)
2
어떻게 감지?
내부 타이머가 마지막 시전 후 180초 경과했는지 계속 체크
3
시전 동작
NumPad8 을 2초 동안 연타 →
버프 OCR 에서 "파력무참" 글자 떴는지 확인 →
실패하면 최대 3번 재시도
재발동: 다시 180초 경과 후 자동
🐯 백호의희원 NumPad 4
1
언제 발동?
쿨 OCR 에서 "백호의희원" 쿨 표시가 안 보일 때 (쿨 끝났다는 뜻)
2
어떻게 감지?
쿨 영역 OCR 이 해당 스킬 쿨 숫자 못 찾으면 "준비됨"으로 판단
3
시전 동작
NumPad4 을 2초 연타 →
쿨 OCR 다시 확인 → 쿨 숫자 나올 때까지 무한 반복
(시전 성공 확인되면 중단)
재발동: 쿨 OCR 값이 다시 0 이 되면 (게임 쿨 만료)
🐯 백호의희원첨 NumPad 5
1
언제 발동?
백호의희원과 동일 (쿨 OCR 에 "백호의희원첨" 쿨 미관측)
2
어떻게 감지?
백호의희원과 동일
3
시전 동작
NumPad5 연타. 쿨 관측까지 무한 반복
재발동: 쿨 OCR 값 0 이 되면
🛡️ 금강불체 NumPad 0
1
언제 발동?
현재 조건 없음. 기본 비활성화 (OFF)
2
어떻게 감지?
자동 감지 경로 없음. 향후 조건 정의 필요
3
시전 동작
외부에서 직접 "금강불체" 요청 있을 때만 NumPad0 연타
재발동: 발동 조건이 코드에 아직 없음 — 원하는 조건 알려주면 추가

공통 구조

  • 검사 주기: HP/MP OCR 은 0.5초마다 값 갱신. 값이 바뀌는 순간 스케줄러 즉시 깨움 (추가 대기 없음)
  • 시전 순서: 여러 조건이 동시에 만족해도 스케줄러 큐에 들어간 순서대로 한 번에 하나씩 시전 (자힐 끝나야 공증 시작)
  • 자힐·자가부활 중 이동: 블록 A+B 시퀀스 진행 중에는 방향키 차단 (TAB/HOME 과 섞이면 꼬임). 끝나면 바로 해제
  • 중복 방지: 같은 스킬이 큐에 이미 있으면 무시. 조건 유지되는 동안 한 번만
  • 발동 정지 조건: armed OFF / 죽음 상태 / UDP 끊김 / 맵 불일치 / 따라가기 전용 모드면 전체 중단
  • UI 임계치 즉시 반영: 스피너 값 변경하면 다음 감지 때 바로 적용 (재시작 불필요)
